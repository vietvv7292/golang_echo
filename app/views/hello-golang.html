👋 Làm quen với Golang

Golang (hay Go) là một ngôn ngữ lập trình mã nguồn mở do Google phát triển, ra mắt vào năm 2009. Golang được thiết kế để giúp các lập trình viên xây dựng ứng dụng nhanh chóng, hiệu quả, dễ bảo trì và có hiệu năng cao.

🚀 Tại sao nên học Golang?

  • Đơn giản và dễ học: Cú pháp của Golang rất rõ ràng, ít từ khóa phức tạp, giúp người mới dễ tiếp cận.
  • Hiệu năng cao: Golang biên dịch trực tiếp xuống mã máy, không cần máy ảo, cho tốc độ thực thi rất nhanh, gần tương đương C/C++.
  • Hỗ trợ Concurrency tuyệt vời: Với goroutines và channels, Golang giúp xây dựng các ứng dụng xử lý đồng thời (đa luồng) rất hiệu quả và dễ dàng hơn nhiều so với các ngôn ngữ khác.
  • Thư viện tiêu chuẩn mạnh mẽ: Bao gồm các package mạng, web, mã hóa, và nhiều hơn nữa, bạn có thể làm gần như mọi thứ mà không cần thư viện bên ngoài.
  • Cộng đồng lớn và liên tục phát triển: Nhiều công ty lớn sử dụng Golang, nên tài liệu, công cụ và thư viện hỗ trợ rất phong phú.

🔤 Cú pháp cơ bản trong Golang

1. Khai báo biến

var a int
var b = "hello"
c := 42

2. Kiểu dữ liệu phổ biến

  • string, int, float64, bool
  • array, slice, map, struct, interface

📦 Giải thích chi tiết các kiểu dữ liệu

1. Kiểu cơ bản (Basic types)

  • string: Chuỗi văn bản. Ví dụ: "Xin chào"
  • int, int8, int16, int32, int64: Số nguyên
  • float32, float64: Số thực (số thập phân)
  • bool: Kiểu logic, chỉ có true hoặc false

2. Mảng và Slice

  • [3]int: Mảng có độ dài cố định gồm 3 số nguyên. Ví dụ: [3]int{1, 2, 3}
  • []int: Slice là một danh sách có độ dài linh hoạt. Ví dụ: []int{4, 5, 6}

3. Map (Bản đồ - Tập hợp khóa/giá trị)

  • Dạng: map[string]int nghĩa là một map với khóa là chuỗi và giá trị là số nguyên.
  • Ví dụ: map[string]int{"apple": 2, "banana": 5}

4. Struct (Cấu trúc)

  • Struct là cách định nghĩa kiểu dữ liệu do người dùng tự định nghĩa.
type Person struct {
  Name string
  Age  int
}

5. Interface (Giao diện)

  • Interface mô tả tập hợp các phương thức mà một kiểu dữ liệu cần có để được coi là "tuân theo" interface đó.
type Speaker interface {
  Speak() string
}

3. Câu lệnh điều kiện

if x > 10 {
  fmt.Println("Lớn hơn 10")
} else if x == 10 {
  fmt.Println("Bằng 10")
} else {
  fmt.Println("Nhỏ hơn 10")
}

4. Vòng lặp for

for i := 0; i < 5; i++ {
  fmt.Println(i)
}

names := []string{"Alice", "Bob"}
for i, name := range names {
  fmt.Println(i, name)
}

5. Hàm

func sayHello(name string) string {
  return "Hello " + name
}

6. Struct

type Person struct {
  Name string
  Age  int
}

p := Person{Name: "Alice", Age: 30}
fmt.Println(p.Name)

📌 Kết luận

Golang là một ngôn ngữ đơn giản, hiện đại và mạnh mẽ, lý tưởng cho việc xây dựng các hệ thống lớn, web service hoặc microservice. Nếu bạn đang tìm kiếm một ngôn ngữ để phát triển hiệu quả, dễ bảo trì và hiệu năng cao – Golang là lựa chọn tuyệt vời.
